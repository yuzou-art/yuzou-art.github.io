### Git

* To sync with the remote repository

  ```
  git pull
  ```

* To update what will be committede

  ```
  git add <file1> <file2> ...
  ```

* To commit

  ```
  git commit -m <comment>
  ```

* To push local `master` branch to the GitHub

  ```
  git push origin master
  ```
